A COMPUTER SYSTEM

HARWARE

A physical parts of a computer that we can touch and feel are called hardware. example monitor,keybord etc.

MAIN PARTS OF A COMPUTER

  • monitor
    1. CRT (Cathode Ray Tube)
    2. LCD ( Liquid Crystal Display)
    3. LED ( Light-Emitting Diode)
  • cpu(central processing unit)
  • keyword
  • mouse

OTHER HARDWARE DEVICES

  • Touch Screen
  • Printer
  • Speaker
  • Pen Drive
  • CD

SOFTWARE

Software is that part of a computer,which cannot be touched.To make the hardware work,we need to have software installed in it.

Type of software

  1. System Software

    2. It is a set of program that controls the overall working of a computer. For Example Operating System

  2. Application software

    3. The software that help you to do specific type of work is called application software. For Example Microsoft Word,Power Point,MS Paint
